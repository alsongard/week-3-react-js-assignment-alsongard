Alson Task Manager API

This API allows you to manage tasks by creating, reading, updating, and deleting them. It is a RESTful interface designed for use in web or mobile applications. All data is served and accepted in JSON format.

üìå Create a Task

Endpoint: POST /task

Full URL: https://alson-task-api.vercel.app/task

This route allows you to create a new task by sending a JSON object in the request body.

Request Body Example:

        {
            task_name: "Complete project",
            task_description: "Finish the frontend and backend",
            task_startDate: "",
            task_endDate: "",
            status: "upcoming/ongoing",
            priority: 1-10, 
            assignee: "Hohn",
        }

Response: Returns the newly created task object with an ID.

üìã View All Tasks

Endpoint: GET /tasks

Full URL: https://alson-task-api.vercel.app/tasks

This route fetches all tasks stored in the database. It returns an array of task objects.

Use Case: Display all tasks in a list or dashboard.

Endpoint: GET /task/:id

Full URL Example: https://alson-task-api.vercel.app/task/64fc8a...

This route returns a single task based on the provided unique ID in the URL.

  • :id ‚Äì The unique identifier of the task (usually MongoDB ObjectID).
